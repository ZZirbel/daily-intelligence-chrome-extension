## üöÄ Setup Instructions

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Python 3.8+ (for news summarization)
- Internet connection (for initial setup and news fetching)

### Step 1: Download Files
Save these files to a permanent location (e.g., `~/Documents/chrome-daily-dashboard/`):
- `chrome-daily-dashboard.html`
- `news-summarizer.py`
- `requirements.txt`
- `.env.example`

### Step 2: Install Python Dependencies
```bash
cd ~/Documents/chrome-daily-dashboard/
pip install -r requirements.txt
```

### Step 3: Get Groq API Key
1. Visit [https://console.groq.com/](https://console.groq.com/)
2. Sign up for a free account
3. Navigate to "API Keys" section
4. Create a new API key
5. Copy the key (it will only be shown once!)

### Step 4: Configure Environment
```bash
# Copy the example file
cp .env.example .env

# Edit .env and add your API key
echo "GROQ_API_KEY=your_actual_api_key_here" > .env
```

### Step 5: Test News Summarizer
```bash
python news-summarizer.py
```

You should see output like:
```
2024-12-19 08:00:00 - INFO - Starting news summarization...
2024-12-19 08:00:05 - INFO - Fetched 10 articles from AI News
2024-12-19 08:00:10 - INFO - Generated summary for ai_news
...
Results saved to summary.json
```

### Step 6: Set Chrome Homepage
1. Open Chrome Settings
2. Go to "On startup"
3. Select "Open a specific page or set of pages"
4. Click "Add a new page"
5. Enter the file path: `file:///Users/YOUR_USERNAME/Documents/chrome-daily-dashboard/chrome-daily-dashboard.html`

### Step 7: Automate Daily News Updates

#### On Mac (using launchd)
Create `~/Library/LaunchAgents/com.aidashboard.news.plist`:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
    <key>Label</key>
    <string>com.aidashboard.news</string>
    
    <key>ProgramArguments</key>
    <array>
        <string>/usr/bin/python3</string>
        <string>/Users/YOUR_USERNAME/Documents/chrome-daily-dashboard/news-summarizer.py</string>
    </array>
    
    <key>WorkingDirectory</key>
    <string>/Users/YOUR_USERNAME/Documents/chrome-daily-dashboard</string>
    
    <key>StartCalendarInterval</key>
    <dict>
        <key>Hour</key>
        <integer>6</integer>
        <key>Minute</key>
        <integer>0</integer>
    </dict>
    
    <key>StandardOutPath</key>
    <string>/Users/YOUR_USERNAME/Documents/chrome-daily-dashboard/news.log</string>
    
    <key>StandardErrorPath</key>
    <string>/Users/YOUR_USERNAME/Documents/chrome-daily-dashboard/news.error.log</string>
</dict>
</plist>
```

Load the agent:
```bash
launchctl load ~/Library/LaunchAgents/com.aidashboard.news.plist
```

#### On Windows (using Task Scheduler)
1. Open Task Scheduler
2. Click "Create Basic Task"
3. Name: "AI Dashboard News Update"
4. Trigger: Daily at 6:00 AM
5. Action: Start a program
   - Program: `python`
   - Arguments: `C:\Users\YOUR_USERNAME\Documents\chrome-daily-dashboard\news-summarizer.py`
   - Start in: `C:\Users\YOUR_USERNAME\Documents\chrome-daily-dashboard\`
6. Finish and test the task

#### On Linux (using cron)
```bash
# Edit crontab
crontab -e

# Add this line (runs daily at 6:00 AM)
0 6 * * * cd ~/Documents/chrome-daily-dashboard && /usr/bin/python3 news-summarizer.py >> news.log 2>&1
```

## üîß Troubleshooting

### News summaries aren't showing
**Problem**: `summary.json` file not found or outdated

**Solution**:
```bash
cd ~/Documents/chrome-daily-dashboard/
python news-summarizer.py
```

Check that `summary.json` was created and contains data.

### Python script fails with "Module not found"
**Problem**: Dependencies not installed correctly

**Solution**:
```bash
pip install --upgrade -r requirements.txt
```

### Groq API errors
**Problem**: Invalid API key or rate limit exceeded

**Solution**:
1. Verify `.env` file contains correct API key
2. Check Groq console for rate limits: [https://console.groq.com/](https://console.groq.com/)
3. Free tier limits: 30 requests/minute, 14,400 requests/day

### Dashboard not loading in Chrome
**Problem**: Incorrect file path or permissions

**Solution**:
1. Verify file path is correct: Right-click HTML file ‚Üí "Get Info" (Mac) or "Properties" (Windows)
2. Check file permissions: `chmod 644 chrome-daily-dashboard.html`
3. Try opening directly: Drag HTML file into Chrome window

### localStorage data lost
**Problem**: Browser cleared cache or incognito mode

**Solution**:
- Export reflections before clearing browser data
- Avoid using incognito mode for dashboard
- Consider implementing cloud backup (future enhancement)

### Scheduled task not running
**Mac**: Check launchd status
```bash
launchctl list | grep com.aidashboard.news
# Should show the service

# Check logs
tail -f ~/Documents/chrome-daily-dashboard/news.log
```

**Windows**: Check Task Scheduler history
1. Open Task Scheduler
2. Find "AI Dashboard News Update"
3. View "History" tab for execution logs

**Linux**: Check cron logs
```bash
grep CRON /var/log/syslog | grep news-summarizer
```

## üìä Data Persistence

All user data is stored in browser localStorage:
- **Seen content tracking**: Ensures no repeats until full cycle
- **Quiz progress**: Spaced repetition intervals and mastery levels
- **Reflections**: All saved reflections with timestamps
- **Preferences**: Theme, collapsed sections, read articles

**Export Your Data**:
```javascript
// Open browser console on dashboard page
console.log(localStorage.getItem('dashboard_state'));
```

**Import Data** (after browser reset):
```javascript
localStorage.setItem('dashboard_state', 'YOUR_EXPORTED_JSON_HERE');
```
